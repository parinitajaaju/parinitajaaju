Prior to this course, I had little to no knowledge about programming. However, this course has helped me delve into a lot of different coding languages, websites, and ideas. This is some of the work I did for the course.

Scratched: Zombie Apocalypse Survival

For my first scratched assignment, I created a Zombie Apocalypse survival game that gives you scenarios and options to click on. It is up to you to choose the right options in order to survive, or else you die.

p5.JS

We used https://editor.p5js.org/ to create our design/animation. For mine, I wanted to create an endless loop. There’s a large square which is the boundary. It starts with one square, and hovering above a square makes it move. Every time the square touches the boundary, it creates a new square until there’s an infinite number of squares in the boundary.

Python: Design Analyser

For our third assignment, we had to create a tool using python that solves a problem. My program helps you analyse a pdf and give you design suggestions. Many times, when you are designing something or creating something on Canva, for example, you forget to check if the font sizes and image quality are consistent. The code analyses your pdf, checks for font and image quality consistency and also gives you other suggestions that can enhance the aesthetic appeal.
